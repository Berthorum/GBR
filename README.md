# GBR - Games Bug Reports

### GBR, acronym for Games Bug Reports, is a personal project that gathers Bug Reports voluntarily made for Digital Games. All games listed in this repository, and their respective bug reports, have been sent to their developers/companies and authorized by them to be used by me as portfolio pieces.
